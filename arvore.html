Estrutura de Dados | Árvore

É uma estrutura de dados que organiza seus elementos de forma hierárquica, onde fica um elemento no topo da árvore, chamado raiz e elementos subordinados a ele, que são chamadas de sub-árvores e folhas

Características de uma Árvore

  • Raiz: Toda árvore tem um nó raiz que é o nó inicial de uma árvore;
  • Grau: o número de filhos que uma árvore possui;
  • Nivel (profundidade): a distância de um nó até a raiz;
  • Altura: o maior nivel da árvore (altura de uma árvore com n nós pode variar de lg(n) até n-1;);
  • folha: o nó que não possui filhos;

Operações em uma Árvore

Com relação às operações em árvores binárias, serão abordadas as seguintes:

  • Inserir um nó na árvore;
  • Removê-lo um nós de uma árvore;
  • Consultar os nós da árvore em ordem;
  • Consultar em pré-ordem;
  • Consultar em pós-ordem;

Árvores Binarias

Considerando uma estrutura de dados Árvore com um conjunto de nós temos que considerar que esse conjunto pode ter outros subconjuntos chamados de sub-árvores que contem cada uma seu próprio conjunto de nós.

O conjunto de nós de uma árvore pode estar vazio ou ser dividido em três subconjuntos distintos, sendo eles: 1º subconjunto (nó raiz), 2º subconjunto (sub-árvore à direita) e 3º subconjunto (sub-árvore à esquerda).

Operações em uma árvore binária

  • Inserir novo elemento
  • Buscar um elemento
  • Mostrar todos os elementos
  • Remover um elemento
  • Contar elementos
  • Calcular nível de um elemento

Árvore em PHP SPL

A classe SplHeap da SPL é uma classe abstrata que implementa uma série de funcionalidades de uma Heap, sendo uma classe abstrata para usar uma estrutura Heap da SPL é preciso usar as classes concretas SplMaxHeap e SplMinHeap. Para inserir um item usamos a operação insert.

SplMaxHeap

SplMaxHeap para definir a posição do valor a ser inserido na árvore ela mantém sempre o maior valor como raiz da árvore, assim apresentando os valores com uma ordenação decrescente

            
              <?php
              $heap = new SplMaxHeap();
              $heap->insert([22,333]);
              $heap->insert([2,33]);
              $heap->insert([222,3]);
              
              var_export($heap->extract());
              echo '
';
              var_export($heap->extract());
              echo '
';
              var_export($heap->extract());
              ?>

A saída apresentada será [222, 3], [22, 33], [2, 33] .

SplMinHeap

A SplMinHeap ao contrário da SplMaxHeap mantém o menor valor como raiz da Árvore, apresentando os valores com ordenação crescente.

            
              <?php
              $heap = new SplMinHeap();
              $heap->insert([22,333]);
              $heap->insert([2,33]);
              $heap->insert([222,3]);
              
              var_export($heap->extract());
              echo '
';
              var_export($heap->extract());
              echo '
';
              var_export($heap->extract());
              ?>

A saída apresentada será [2, 33] , [22, 33], [222, 3] .
