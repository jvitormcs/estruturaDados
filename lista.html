Estrutura de dados | Lista

Sabemos da importância da estrutura de dados para a construção do código. Mas afinal, o que é lista?

Conceito

  • A estrutura de dados do tipo lista, armazena os dados de um determinado tipo em uma ordem específica
  • A diferença entre lista e arrays é que a lista possui um tamanho ajustável, enquanto arrays possui um valor fixo
  • Existem dois tipos de lista sendo elas as ligadas e as duplamente ligadas
  • Na estrutura do tipo lista ligada existem os nós onde cada um dos nós conhece o valor que está sendo armazenado em seu interior, além de conhecer o valor posterior a ele: por isso ela é chamada de “lista ligada”, pois os nós são amarrados com essa indicação de qual é o próximo nó
  • A lista duplamente ligada é uma variação da lista ligada
  • A diferença da lista duplamente ligada e a lista ligada é que a lista duplamente ligada são bidirecionais. Vimos que, naturalmente, não conseguimos “andar para trás” em listas ligadas, já que, os nós das listas ligadas somente sabem quem é o próximo elemento. Já nas listas duplamente ligadas os nós conhecem o próximo elemento, como também sabem qual é seu elemento anterior, o que nos permite a navegação reversa

Exemplo

A SplDoublyLinkedList é a lista duplamente ligada, como vimos, são bidirecionais. A operação unshift insere no início da lista: 
            
              <?php $lista = new SplDoublyLinkedList();
              $lista->unshift("Beatriz");
              $lista->unshift("Paulo");
              $lista->unshift("Roberto");
A push insere no final da lista. Para exemplificar, vamos usar a lista criada anteriormente de nomes e adicionar mais uma pessoa ao final dela: 
            
              <?php $lista->push("Laura");
O shift por sua vez, remove um item do início da lista: 
                        
                            <?php $lista→shift(); // Removeu Beatriz
O pop por sua vez, remove um item do final da lista: 
            
              <?php $lista→pop(); // Removeu Laura
