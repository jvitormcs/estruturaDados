Estrutura de dados

Conceito

Estrutura de dados é uma estrutura organizada de dados na memória de um computador ou em qualquer dispositivo de armazenamento (pen drive, hds externos, nuvem etc.), de forma que os dados possam ser utilizados de forma correta.

Essas estruturas encontram muitas aplicações no desenvolvimento de sistemas sendo que algumas são altamente especializadas e utilizadas em tarefas específicas.

Usando as estruturas adequadas através de algoritmos, podemos trabalhar com grandes e pequenas quantidades de dados, como aplicações em banco de dados ou serviços de buscas.

  • Vetores e Matrizes (Arrays)
