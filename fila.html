Estrutura de dados | Fila

Sabemos da importância da estrutura de dados para a construção do código. Mas afinal, o que é fila?

Conceito

A Fila (função SplQueue) tem o princípio FIFO (first in, first out), ou seja, os elementos que são inseridos primeiro, serão os primeiros a serem removidos.

Exemplo

Adicionando na fila de ingredientes: 

                    
                    <?php $ingredientes=new splQueue(); $ingredientes -> enqueue ('Cenoura');
                            $ingredientes -> enqueue ('Batata');
                            $ingredientes -> enqueue ('Pimentão');?>

Removendo da fila de ingredientes 

                    
                    <?php echo $ingredientes->dequeue(); // Remove Cenoura
                            echo $ingredientes->dequeue(); // Remove Batata
                            ?>

Verificar se a fila está vazia

                    
                    <?php $ingredientes = SplQueue();
                    echo $ingredientes-> isEmpty();
